# Student-Marks-Analyzer-in-R
A menu-driven R program to analyze student marks with totals, averages, pass/fail, topper detection, graphs, and CSV export.

## ‚ú® Features
- Dynamic input of students and subjects
- Calculates **Total** and **Average** marks per student
- Identifies **Pass/Fail** students
- Finds the **Topper**
- Subject-wise and student-wise analysis
- Graphs:
  - Bar plot of student totals
  - Bar plot of subject averages
  - Pie chart of pass vs fail
- Option to save results to CSV file
